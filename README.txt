  Password Checker

   Описание
Этот проект анализирует сложность пароля, проверяет его на утечки через API "Have I Been Pwned" и выявляет распространённые слабые места пароля.

   Возможности
- Проверка пароля на наличие в базе утечек.
- Оценка сложности пароля на основе длины, наличия заглавных букв, цифр и спецсимволов.
- Выявление распространённых шаблонов, таких как повторяющиеся символы или частые слова.
- Использование списка из 10 000 самых популярных паролей для сравнения.

   Установка и запуск

    1. Установите зависимости
Этот проект использует Python 3. Установите необходимые библиотеки:

pip install requests


    2. Запуск анализа пароля
Запустите скрипт:

python password_checker.py


   Структура проекта
- 'password_checker.py' - Основной файл, запрашивающий ввод пароля и запускающий анализ.
- 'validators.py' - Функции проверки пароля, включая анализ структуры пароля и сравнение с базой популярных паролей.
- 'password_check_api.py' - Взаимодействие с API "Have I Been Pwned" для проверки пароля на утечки.
- '10k-most-common.txt' - Файл со списком 10 000 самых распространённых паролей.
